# Crypto Paper Trading Bot 🚀

A Telegram bot for cryptocurrency paper trading that allows users to simulate trading without using real money.

The bot uses Binance API to retrieve cryptocurrency prices and provides basic trading functionality through Telegram commands.

## Features

- Real-time cryptocurrency price tracking
- Virtual buy and sell operations
- Portfolio management
- Trade history
- Binance API integration
- Telegram bot interface

## Tech Stack

- JavaScript
- Node.js
- Telegram Bot API
- Binance API
- REST API

## Installation

bash
```
npm install
```
Run
```
npm start
```
